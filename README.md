# Simple Item Tracker

A lightweight, stateful mobile application built with Flutter. This project demonstrates core Flutter concepts including local state management, text input handling, and dynamic list rendering.

## Features
* **Create:** Add new items to a dynamic list using a text controller.
* **Read:** View all items in a scrollable `ListView` utilizing Material Design cards.
* **Update:** Select existing items to modify their content dynamically.
* **Delete:** Remove items from the list with immediate UI updates.

## Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **UI/UX:** Material Design components (`Scaffold`, `AppBar`, `ListView.builder`, `Card`)

## Getting Started

To run this project locally, ensure you have the Flutter SDK installed.

1. Clone the repository:
   `git clone https://github.com/devnathm/simple-list-app.git`
2. Navigate to the project directory:
   `cd simple-list-app`
3. Install dependencies:
   `flutter pub get`
4. Run the app:
   `flutter run`
